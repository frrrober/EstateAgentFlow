# EstateAgentFlow — Framework Multi-Agents

## Présentation
EstateAgentFlow est un framework open source conçu pour orchestrer le parcours complexe de l'achat d'une maison neuve. Ce système déploie des agents IA spécialisés pour analyser vos contrats, vos risques financiers et les étapes techniques de votre construction.

## Mission Sociale
Au-delà de l'immobilier, ce framework est un prototype pour l'accessibilité numérique. Conçu pour réduire la charge cognitive et structurer l'information, cet outil vise à soutenir les étudiants et travailleurs dyslexiques. En automatisant la mise en forme de projets complexes, ce framework offre une autonomie précieuse à ceux qui ne bénéficient pas d'aides institutionnelles.

## Fonctionnalités
* **Analyse de Documents :** Analyse automatique des contrats et spécifications techniques.
* **Évaluation des Risques :** Identification des points critiques du calendrier d'acquisition.
* **Gestion de Projet :** Suivi automatisé des étapes clés.
* **Aide à la Décision :** Conseils structurés pour sécuriser votre projet.
